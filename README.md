### ❓ 질문 사항

#### 1. string 네이밍 방법
뷰모델로 데이터를 연결하기 전에, TextView에 표시될 힌트 문구를 string.xml로 관리하고자 합니다. 이때, 이러한 문구들의 이름을 어떻게 지어야 할지 고민이 됩니다. 멘토님께서는 보통 문자열 리소스의 네이밍을 어떻게 하시는지 여쭤보고 싶습니다.
